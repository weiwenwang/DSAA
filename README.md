### 三、使用方法

#### 1. 编译

    进入到代码目录

    cd DSAA/xxx/code(eg: cd DSAA/intSLList/code)

    清除上次的make产生的可执行文件

    ./build.sh clean

    编译

    ./build.sh


#### 2. 执行代码

    ../build/bin/main
